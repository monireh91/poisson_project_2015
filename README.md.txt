# Poisson Regression Project (2015)

This is a simple educational example of Poisson regression in Base R.
It includes:
- Generating Poisson-distributed data
- Descriptive statistics (mean, variance)
- Barplot of data distribution
- Fitting a Poisson regression model
- Predicting values and comparing actual vs predicted
- Plotting results
